# Libman - Library manager app

![License](https://img.shields.io/badge/license-MIT-blue)
![Status](https://img.shields.io/badge/status-In%20Development-orange)
![Tech Stack](https://img.shields.io/badge/stack-React%20%7C%20Node.js-informational)

Libman is a full-stack web application for manage essential operations for a modern library. It provides an easy way to browse catalogs and manage loans, while offering secure, role-based dashboards for librarians and administrators to manage inventory and user accounts.

## Feature

- **Public Acess**
  - **Book Catalog:** Landing page displaying a searchable list of all available books.
  - **My Loans:** Dedicated user page for viewing current loans and due dates.

* **Role-Based Management**
  - **Secure Dashboard:** Access restricted by user role (Librarian or Admin).
  - **Librarian Privilege**
    - **Borrow/Return:** Efficient operations for checking books in and out.
    - **Member Registration:** Quick registration of new library member accounts.
  - **Admin Privilege**
    - **Inventory Management:** Full CRUD (add, edit, delete) capabilities for the book catalog.
    - **User Management:** Full CRUD capabilities for user accounts and roles.
